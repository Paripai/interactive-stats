Systematic Sampling
การสุ่มตัวอย่างแบบมีระบบ

สุ่มจุดเริ่มต้น แล้วเลือกทุกๆ k หน่วย โดย k = N/n

ขั้นตอน
  1. คำนวณ k = N ÷ n
  2. สุ่มจุดเริ่มต้น r (1 ถึง k)
  3. เลือก r, r+k, r+2k, … จนครบ
✓ ข้อดี
  • ง่าย รวดเร็ว
  • กระจายครอบคลุม list
✗ ข้อจำกัด
  • อาจมี periodic bias
  • ต้องเรียงรายการก่อน

การสุ่มตัวอย่างแบบแบ่งชั้น

แบ่ง population เป็น strata แล้วสุ่มจากแต่ละกลุ่มแยกกัน

ขั้นตอน
  1. กำหนด strata ตามตัวแปรที่สนใจ
  2. คำนวณสัดส่วนแต่ละ strata
  3. สุ่มจากแต่ละ strata ตามสัดส่วน
✓ ข้อดี
  • ตัวแทนครบทุกกลุ่ม
  • แม่นยำสูงกว่า SRS
✗ ข้อจำกัด
  • ต้องรู้ข้อมูล strata ก่อน
  • ซับซ้อนกว่า SRS

Cluster Sampling

ขั้นตอน
  1. แบ่ง population เป็น cluster
  2. สุ่มเลือก m cluster
  3. เก็บข้อมูลทุกหน่วยใน cluster ที่เลือก
✓ ข้อดี
  • ประหยัดค่าเดินทาง
  • ไม่ต้องมี frame ทั้ง pop.
✗ ข้อจำกัด
  • Sampling error สูงกว่า
  • หน่วยในกลุ่มอาจคล้ายกัน

  3. หยุดเมื่อครบ n

  • รวดเร็ว ประหยัด
  • เหมาะกับ pilot study

  • ไม่สามารถ generalize

Purposive Sampling
การสุ่มตัวอย่างแบบเจาะจง

ผู้วิจัยใช้วิจารณญาณเลือกหน่วยที่ตรงเกณฑ์ที่กำหนด

ขั้นตอน
  1. กำหนดเกณฑ์คัดเลือก
  2. ระบุหน่วยที่ตรงเกณฑ์
  3. เลือกทั้งหมดหรือสุ่มจากกลุ่มนั้น
✓ ข้อดี
  • ได้หน่วยที่ใช่จริงๆ
  • เหมาะวิจัยเชิงคุณภาพ
✗ ข้อจำกัด
  • อาศัย judgment ผู้วิจัย
  • ไม่สามารถ generalize

Quota Sampling
การสุ่มตัวอย่างแบบโควตา

กำหนดโควตาแต่ละกลุ่มล่วงหน้า แล้วเลือกหน่วยจนครบโควตา คล้าย Stratified แต่ไม่สุ่ม

ขั้นตอน
  1. กำหนดกลุ่มย่อย (quota cells)
  2. กำหนดจำนวนที่ต้องการแต่ละกลุ่ม
  3. เก็บข้อมูลจนครบโควตา
✓ ข้อดี
  • มีสัดส่วนกลุ่มตามที่กำหนด
  • ไม่ต้องมี sampling frame
✗ ข้อจำกัด
  • ภายในกลุ่มยังมี bias
  • ขึ้นกับผู้สัมภาษณ์

Snowball Sampling
การสุ่มตัวอย่างแบบลูกหิมะ

เริ่มจาก seed แล้วขอให้แนะนำผู้รู้จักต่อไปเรื่อยๆ เหมาะกับประชากรที่หาตัวยาก

ขั้นตอน
  1. เลือก seed 1–3 คน
  2. ขอให้แนะนำผู้รู้จักที่ตรงคุณสมบัติ
  3. ทำซ้ำจนครบ n หรืออิ่มตัว
✓ ข้อดี
  • เข้าถึง hidden population
  • สร้างความไว้วางใจผ่าน referral
✗ ข้อจำกัด
  • ขึ้นกับ network ของ seed
  • ไม่ generalize ได้
